if you are a developer who is looking to contribute to Swirl core then read this guide. It will guide you through on how to setup your environment to develop Swirl.

## compiling from source
Follow the steps below to compile the source

### requirements

-   A Linux based distro or Windows
-   Git (version >= 2.35)
-   GCC compiler (version >= 6.1, C++ >= 14) or tdm-gcc on Windows
-   CMake (version >= 3.5)

Install the required build dependencies and programs:

### Linux

<Tabs>
<TabItem value="ubuntu" label="Ubuntu">

```shell
sudo apt install git build-essentials cmake
```

</TabItem>
<TabItem value="fedora" label="Fedora">

```shell
sudo dnf install gcc git cmake
```

</TabItem>
<TabItem value="arch" label="Arch">

```shell
pacman -S git gcc cmake
```

</TabItem>
</Tabs>

### Windows
| Program      | Link      |
|--------------|-----------|
| Git | [git-scm.com](http://git-scm.com/download/win)      |
| gcc      | [tdm-gcc.org](https://jmeubank.github.io/tdm-gcc)|
|CMake|[cmake.org](https://cmake.org/download/#latest)|
MinGW-w64 should also work but we haven't tested it.

### Clone the repo

```shell
git clone https://github.com/SwirlLang/Swirl
```

It will create a folder called Swirl.

We recommended vscode for the development of Swirl. But if you prefer to do it from the command line go to [command line](#command-line) section

## Compiling

### Visual Studio Code
`cd` into the cloned folder and open it in vscode using `code .`

install the required extensions.
-   C/C++ language support by Microsoft
-   CMake Tools by Microsoft

press ctrl+p and run the following command
```
ext install ms-vscode.cpptools
```
```
ext install ms-vscode.cmake-tools
```

#### setup cmake
create a folder in the project directory named .vscode. Inside it create a file named settings.json and paste the following content

```json
{
    "cmake.sourceDirectory": "${workspaceFolder}/Swirl"
}
```

There should be a prompt asking you to configure the project if not run the `CMake: Configure` command in the command pallate.

now click this play button to build and run the binary

![run the app](/img/run-button.png)

the terminal should show this help message. If it did then you have succesfully compiled Swirl.

![terminal output](/img/terminal-output.png)

### Command line

`cd` into the cloned folder and follow the steps below.

#### Configure CMake
```bash
cmake -B build -DCMAKE_BUILD_TYPE=Debug -S Swirl
```

#### Build the project
<Tabs groupId="os">
<TabItem value="linux" label="Linux">

```bash
cmake --build build --config Debug
```

</TabItem>
<TabItem value="windows" label="Windows">

```bash
cmake --build build --config Debug -G "MinGW Makefiles"
```

</TabItem>
</Tabs>

#### Run the app
<Tabs groupId="os">
<TabItem value="linux" label="Linux">

```shell
./build/swirl
```

</TabItem>
<TabItem value="windows" label="Windows">

```shell
.\build\swirl
```

</TabItem>
</Tabs>


It should show the help message if it did then it has been compiled succesfully.

![terminal output - Swirl help](/img/terminal-output.png)
