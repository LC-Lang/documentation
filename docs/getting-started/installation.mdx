Swirl is available on all major platforms. Linux, macOS, and Windows.

## Requirements

A C++ compiler is needed for Swirl to compile the programs. Read the guide on how to [install a C++ compiler](install-cpp-compiler.mdx).


## Download

Download the latest verson of Swirl from the [website](https://swirl-lang.vercel.app/download) or the [GitHub](https://github.com/SwirlLang/Swirl/releases) release page.

On Linux and macOS, you may need to make the binary executable with `chmod +x {binary}`.

Test to see if its working.

<Tabs groupId="os">
<TabItem value="linux" label="Linux">

```shell
./swirl-linux --version
```

</TabItem>
<TabItem value="macos" label="macOS">

```shell
./swirl-macos --version
```

</TabItem>
<TabItem value="windows" label="Windows">

```shell
swirl-windows.exe --version
```

</TabItem>
</Tabs>

## Source

If you want to compile Swirl from source, follow the steps below.

### Requirements

-   Linux, macOS, or Windows
-   Git (version 2.35 and higher)
-   C++ compiler (C++ 14 and higher)
-   CMake (version 3.5 and higher)

Read the [guide](install-cpp-compiler.mdx) on installing a C++ compiler.

Install the required build dependencies and programs:

### Linux

<Tabs>
<TabItem value="ubuntu" label="Ubuntu">

```shell
sudo apt install git cmake
```

</TabItem>
<TabItem value="fedora" label="Fedora">

```shell
sudo dnf install git cmake
```

</TabItem>
<TabItem value="arch" label="Arch">

```shell
sudo pacman -S git cmake
```

</TabItem>
</Tabs>

### Windows
```shell
choco install git cmake
```

### Clone the repo

```shell
git clone https://github.com/SwirlLang/Swirl
```

It will clone into a folder called Swirl.

### Building

### Visual Studio Code

`cd` into the cloned folder and open it in vscode using `code .`

install the required extensions: 

press <kbd>ctrl+p</kbd> and run the following command
```
ext install ms-vscode.cmake-tools
```

#### Setup cmake
create a folder in the project directory named .vscode. Inside it create a file named settings.json and paste the following content

```json
{
    "cmake.sourceDirectory": "${workspaceFolder}/Swirl"
}
```
Now configure the project using the `CMake: Configure` from the command pallate.

now click this play button to build and run the binary

![run the app](/img/run-button.png)

the terminal should show this help message. If it did then Swirl has been compiled successfully.

![terminal output](/img/terminal-output.png)

### Command line

`cd` into the cloned folder and follow the steps below.

#### Configure CMake
<Tabs groupId="os">
<TabItem value="linux" label="Linux">

```bash
cmake -B build -DCMAKE_BUILD_TYPE=Debug -S Swirl
```

</TabItem>
<TabItem value="windows" label="Windows">

```bash
cmake -B build -DCMAKE_BUILD_TYPE=Debug -S Swirl -G "MinGW Makefiles"
```

</TabItem>
</Tabs>

#### Build the project
```bash
cmake --build build --config Debug
```

#### Run the app
<Tabs groupId="os">
<TabItem value="linux" label="Linux">

```shell
./build/swirl
```

</TabItem>
<TabItem value="windows" label="Windows">

```shell
.\build\swirl.exe
```

</TabItem>
</Tabs>


It should show the help message if it did then it has been compiled succesfully.

![terminal output - Swirl help](/img/terminal-output.png)
