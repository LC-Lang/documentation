Swirl is available on Linux and Windows.

## Linux

Swirl is available on all major distributions of linux. Debian/Ubuntu, Fedora and Arch

### Debian/Ubuntu {#debian-ubuntu}

Debian 10 or Ubuntu 20.04 or higher is required.

```shell
sudo apt install Swirl
```

### Fedora

Fedora 34 or higher is required.

```shell
sudo dnf install Swirl
```

### Arch Linux

```shell
sudo pacman -S Swirl
```

## Windows

Swirl has been tested on Windows 10 only.

You can download the Windows installer from the [download page](https://swirl-lang.vercel.app/download/windows).
After downloading the installer follow the instructions to install Swirl.

After installing Swirl you can run it from cmd prompt.

```shell
swirl test.sw
```
