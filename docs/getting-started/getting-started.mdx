## Introduction

Swirl is a programming language that focuses on writing <b>elegant</b> and <b>readable</b> code. That can maximize your productivity and minimize effort.

Some of the key features of Swirl:
-   Easy learning curve
-   Clean and concise syntax
-   Efficient apps
-   Available for all major platforms
-   Powerful set of standard libraries
-   Awesome package management

## Get started

Get started with **Swirl in just 5 minutes**

### Installation

Check out the [installation](installation) page for more options of downloading. Here only linux is shown.

Assuming you are using a debian, fedora or a Arch based distro use the below commands to install Swirl:

<Tabs>
<TabItem value="debian" label="Debian">

```shell
sudo apt install Swirl
```

</TabItem>
<TabItem value="fedora" label="Fedora">

```shell
sudo dnf install Swirl
```

</TabItem>
<TabItem value="arch" label="Arch">

```shell
sudo pacman -S Swirl
```

</TabItem>
</Tabs>


It will install Swirl on your system.

It will contain the Swirl compiler and the standard libraries.

### Compile a file

create a file for compiling it. Use your preferred editor.

```shell
nano test.sw
```

Then paste the following in the file or write your own

```sw title="test.sw"
print("Hello World!, I am learning Swirl")
```

now compile the file

```shell
swirl test.sw
```

It will make a binary called **test** in the current folder. Execute it using `./test`.

It should output `Hello World!, I am learning Swirl`.

Congrats you've made your first Swirl app.
